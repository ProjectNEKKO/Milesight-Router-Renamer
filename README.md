## 🚀 Getting Started  
Follow these steps to set up and run the application on your local machine.

1. **Prerequisites**  
   Make sure you have Python 3.8 or higher installed on your system. You can check your version by running:  
   ```bash
   python --version
   ```  
2. **Clone the Repository**  
   Clone this repository to your local machine:  
   ```bash
   git clone https://github.com/ProjectNEKKO/Milesight-Router-Renamer.git  
   cd Milesight-Router-Renamer  
   ```  
3. **Install Dependencies**  
   Install Streamlit using pip:  
   ```bash
   pip install streamlit  
   ```

## 🖥️ How to Run the App
Open your terminal or command prompt inside the project folder.  

Launch the Streamlit server:  
```bash
python -m streamlit run renamer.py
```
